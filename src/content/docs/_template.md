# Título Principal del Documento

Aquí empieza el contenido del documento. Escribe de forma clara y organizada.

## Sección Principal

Usa títulos ## para las secciones principales. Estos aparecerán con una línea de color debajo.

### Subsección

Usa títulos ### para subsecciones más pequeñas.

## Formateo de texto

Puedes usar **negrita** para destacar texto importante y *cursiva* para énfasis.

## Listas

Puedes hacer listas así:

- Primer elemento
- Segundo elemento
- Tercer elemento

O listas numeradas:

1. Paso uno
2. Paso dos
3. Paso tres

## Código

Para mostrar código inline usa `código así`.

Para bloques de código más largos:

```javascript
const ejemplo = "Hola mundo";
console.log(ejemplo);
```

## Citas importantes

> Usa esto para destacar notas importantes o advertencias.
> Las citas se verán con un borde de color a la izquierda.

## Enlaces

Puedes añadir [enlaces a otros sitios](https://ejemplo.com) o mencionar canales de Discord como <#canal-ejemplo>.

**CONSEJOS PARA STAFFS:**

1. **Título y descripción son OBLIGATORIOS** - El resto es opcional
2. **Para subir imágenes:** Colócalas en `/public/images/` y luego usa `/images/nombre.png`
3. **Color:** Elige el que mejor combine con el tema del doc
4. **Formato:** Usa ## para secciones grandes, ### para subsecciones
5. **No te compliques:** Escribe de forma natural, como si explicaras a un amigo

**Estructura recomendada:**
- Introducción breve explicando de qué va el doc
- Secciones organizadas con títulos claros
- Ejemplos o pasos cuando sea necesario
- FAQ al final si hace falta
